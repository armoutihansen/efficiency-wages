# Data

The analysis uses these treatment-level input files:

- `raw/ge_prosocial/combined.csv`: GE and prosocial treatments from the original joint export.
- `raw/neutral/combined.csv`: neutral treatment.
- `raw/efficiency/combined.csv`: efficiency treatment.

`derived/` is generated by `analysis.ipynb` or `uv run python run_analysis.py`. It is ignored by git because it can always be rebuilt from the raw inputs.

Treatment mapping:

- `P`: GE treatment.
- `S`: Prosocial treatment.
- `N`: Neutral treatment.
- `PAN`: Efficiency treatment.
